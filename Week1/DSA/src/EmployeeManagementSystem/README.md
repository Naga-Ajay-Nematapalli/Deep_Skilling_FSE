# Employee Management System

## Overview

The **Employee Management System** is a Java-based application that demonstrates the use of arrays to manage employee records. It provides basic CRUD operations such as adding, searching, displaying, and deleting employee details while analyzing the efficiency of each operation.

## Features

* Add new employee records
* Search employees using Employee ID
* Display all employee records
* Delete employee records
* Analyze time complexity of operations

## Technologies Used

* Java
* Arrays
* Object-Oriented Programming (OOP)

## Project Structure

```
EmployeeManagementSystem/
├── Employee.java
├── EmployeeManagementSystem.java
├── Main.java
└── README.md
```

## Employee Attributes

* Employee ID
* Name
* Position
* Salary

## Operations Implemented

* Add Employee
* Search Employee
* Display Employees
* Delete Employee

## Time Complexity

| Operation         | Complexity |
| ----------------- | ---------- |
| Add Employee      | O(1)       |
| Search Employee   | O(n)       |
| Display Employees | O(n)       |
| Delete Employee   | O(n)       |

## Limitations

* Fixed-size array
* Linear search for employee records
* Deletion requires shifting elements
* Cannot dynamically resize the array

## Learning Outcomes

* Understanding array representation in memory
* Applying Object-Oriented Programming concepts
* Implementing CRUD operations using arrays
* Analyzing algorithm time complexity
* Understanding the advantages and limitations of arrays
